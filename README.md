# react-native-challenge

## Getting Started

Clone the repo:
```sh
git clone https://github.com/thetevikrant/react-native-challenge.git
cd react-native-challenge
```

Install dependencies:
```sh
npm install
```

Start server:
```sh
# Start server
npm start


Tests:
```sh
# Run tests written in ES6 
npm run test
```

Lint:
```sh
# Lint code with ESLint
npm run lint

# Fixing the lint issue
npm run lint:fix
```
